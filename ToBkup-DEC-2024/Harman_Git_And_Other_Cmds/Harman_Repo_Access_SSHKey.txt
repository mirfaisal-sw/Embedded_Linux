

To generate SSH key follow below URL:-
https://download.asperasoft.com/download/docs/ascp/3.5.2/html/dita/creating_public_key_cmd.html

Step I.
=>Command to genrate SSH key: 
$ssh-keygen -t rsa

Step II.
=> Read public key from file - ~/.ssh/id_rsa.pub
$vim id_rsa.pub

Copy entire content of file.

StepIII.
=> SSH key can be added in following URL for example: -
  https://androidhub.harman.com/settings/#SSHKeys

  After opening above URL, click on "SSH keys" button, then paste the copied content
  in box - "New SSH keys".

  Then click on the button "Add new SSH Key".
