

PCI Endpoint Test driver
============================
modulename: pci_endpoint_test.ko
configname: CONFIG_PCI_ENDPOINT_TEST
Linux Kernel Configuration
└─> Device Drivers
	└─> Misc devices
		└─> PCI support
			└─> PCI Endpoint Test driver

Enable this configuration option to enable the host side test driver
for PCI Endpoint.

source code:
drivers/misc/pci_endpoint_test.c

depends -> CONFIG_PCI
selects -> CONFIG_CRC32


Typical config macros to enable: -
=====================================
The following config options have to be enabled in order to configure the PCI controller to be used as a “Endpoint Test” function driver.

CONFIG_PCI_ENDPOINT=y
CONFIG_PCI_ENDPOINT_CONFIGFS=y
CONFIG_PCI_EPF_TEST=y
CONFIG_PCI_J721E=y
CONFIG_PCIE_CADENCE_EP=y
