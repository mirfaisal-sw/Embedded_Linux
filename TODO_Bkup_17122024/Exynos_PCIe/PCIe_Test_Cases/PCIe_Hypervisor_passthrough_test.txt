
==========================================================================
Test case: - Verify hypervisor shall support passthrough for PCI devices
==========================================================================


PreCondition: 1. Headunit is flashed with latest binary
	      2. KL30 = ON 
			  
1. Check PCIe is not given passthrough in Node0
--------------------------------------------------
Step I. Go to Node0(Linux) console
		Press (ESC+O+6+Q)
		
Step II. Run below commands.
	# cd /sys/devices/platform
	# ls *.pcie
	O/p - 
	ls: cannot access '*.pcie': No such file or directory
	
2. Check PCIe is given passthrough to Android
---------------------------------------------------
Step I. Go to Android console
		Press (ESC+O+6+R)
	
Step II. Run below commands.
	# cd /sys/devices/platform
	# ls *.pcie
	Expected O/p - 
	161c0000.pcie:
	driver           pci0000:00  supplier:platform:10830000.pinctrl
	driver_override  pcie_sysfs  supplier:platform:16040000.pinctrl
	modalias         power       uevent
	of_node          subsystem

	161c1000.pcie:
	driver           pci0001:00  supplier:platform:10830000.pinctrl
	driver_override  pcie_sysfs  supplier:platform:16040000.pinctrl
	modalias         power       uevent
	of_node          subsystem
