Install Sparse procedure: -


1) Running sparse on Linux Kernel: -
	$make C=1 -j12 zImage > ../sparse.out 2>&1
	$vim  ../sparse.out

	Note : - Here we can even use C=2

2) Running sparse on modules: -
	 $make C=2 modules  >build_1.out 2>&1
	 $vim build_1.out
